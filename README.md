# 🚢 Titanic Survival Prediction

## 📌 Overview

This project analyzes the Titanic dataset to understand the factors affecting passenger survival. It focuses on data cleaning, exploratory data analysis, and extracting meaningful insights from the data.

---

## 📊 Dataset

The dataset is taken from the Kaggle Titanic competition and includes passenger details such as age, gender, class, fare, and survival status.

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## ⚙️ Project Workflow

1. Data Cleaning & Preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Data Visualization
5. Insight Generation

---

## 📈 Exploratory Data Analysis

* Survival rate based on gender
* Survival rate based on passenger class
* Age distribution of passengers
* Correlation between features

---

## 🎯 Results

* Identified key factors affecting survival
* Generated meaningful insights using data visualization

---

## 💡 Key Insights

* Female passengers had higher survival rates
* 1st class passengers were more likely to survive
* Younger passengers had slightly better survival chances
